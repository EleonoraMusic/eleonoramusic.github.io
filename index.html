Education and Job experiences

My background is in the humanities. After a high-school diploma in arts (Liceo Artistico) I continued my path in Art History. During University I started playing with digital graphic software such as Photoshop and Illustrator. To improve my skills I attended a Web Design course at NABA, in Milan.

I love learning new things and Open Online Courses are a great way to improve my skills. Check out my LinkedIn profile to find out more about me.

Projects I'm working on

To complete the FreeCodeCamp Responsive Web Design Certification, I'm working to develop some static web pages, starting from simple and evolving to more complicated and resposive ones. In the meantime, I started aproaching JavaScript and decided to create another little project to put what I've learned into action.

Tribute page

The first challange was to build a simple tribute page. I chose an icon of Italian design: the Lettera22 by Olivetti. Graphic choices were made following the style of the Lettera22: round corners, light colors and a title that imitates the keyboard of the typewriter. This last one thing was my very first aproach in creating shapes with CSS. It was not required to build a responsive page, but I tried to do my best to create a page that looks good on most devices. Still this project can't really be considered responsive.

Survey Form

This second challange was to create a survey form. You will see that the web design is turning a bit more complex, with a background image and a second layer in front of it, accomplished using pseudo element "::before". I tried to do my best to make this project responsive and, in the end, I'm quite satisfied with the outcome.

WebApp "Name2Numebers"

This project was born by a challange launched by my math-lover boyfriend. He asked me to create a quick website to calculate the numerical value of his name. Of course the main issue was to develop the JS code (in fact the "graphic side" was not a priority here). As a newbie I had to make many online researches and that was still not enough to accomplish my final goal: create a two-pages website where data are transferred from the home page to the result page. To resolve the issue I asked for help to a frontend developer friend so this can be definitely considered a 6-hands project.
